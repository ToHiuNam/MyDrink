# MyDrink

一个功能完整的饮品追踪与健康管理应用 MyDrink，
帮助用户记录每日摄入的咖啡、奶茶、酒和水，分析咖啡因、酒精、水分、糖分、热量及脂肪数据，并提供健康建议。
整体采用 React + Tailwind CSS + Recharts 构建，目前版本的用户数据存储在 localStorage 中。

## 本地运行

1. 先安装 Node.js 20+（安装后会有 `node` 和 `npm` 命令）。
2. 在项目目录执行：

```bash
npm install
npm run dev
```

1. 打开终端输出的本地地址。

## 功能特点

### 底部导航

- **记录**：记录和查看当日饮品摄入情况
- **数据趋势**：查看历史数据和健康分析
- **个人设置**：管理个人信息和应用设置

### 记录页

- **饮品快速添加**：咖啡、奶茶、酒、水四大类饮品一键添加
- **详细参数选择**：支持选择杯型、冰度、糖度等参数
- **今日摄入概览**：实时显示咖啡因、酒精、水分、糖分、热量、脂肪的摄入情况
- **健康助手**：提供补水提醒、咖啡因状态、酒精代谢提醒、糖分摄入提醒、热量摄入提醒、脂肪摄入提醒
- **动态卡片**：根据摄入情况显示不同的视觉效果和提醒

### 数据趋势页

- **多维度统计**：支持今日、本周、本月、本年、近7天的数据查看
- **饮品占比图**：饼图展示不同饮品的摄入比例
- **趋势图表**：折线图展示咖啡因、酒精、水分的摄入趋势
- **历史记录**：详细的饮品摄入历史列表
- **日历热图**：直观显示每天的饮品记录活跃度
- **健康洞察**：基于数据的健康建议和趋势分析

### 个人设置页

- **个人信息管理**：头像、昵称、年龄、体重、血糖、性别等基本信息
- **每日摄入设置**：咖啡因、酒精、糖分、热量、脂肪上限和喝水目标
- **个性化设置**：主题选择（浅色/深色）、UI风格选择（默认/像素/苹果风格）、个性化颜色
- **睡眠设置**：通常入睡时间、通常起床时间、睡眠质量、咖啡因敏感度
- **自定义饮品**：创建、编辑、删除自定义饮品
- **配料管理**：管理饮品配料及其营养成分
- **数据管理**：导出CSV记录、一键清空全部记录

### 技术特点

- **响应式设计**：适配不同设备屏幕尺寸
- **深色模式**：支持浅色和深色主题切换
- **本地存储**：数据自动保存到localStorage，刷新页面不丢失
- **动态效果**：平滑的动画和过渡效果
- **直观的可视化**：使用Recharts库提供数据可视化
- **模块化组件**：代码结构清晰，易于维护

## 核心依赖

- **React**：18.3.1 - 前端框架
- **Recharts**：2.15.4 - 数据可视化库
- **Tailwind CSS**：3.4.17 - 实用优先的CSS框架
- **Vite**：5.4.19 - 现代前端构建工具

## 数据存储

- **localStorage**：用于存储用户记录、个人信息、设置等数据
- **数据持久化**：刷新页面后数据不会丢失
- **数据导出**：支持导出CSV格式的记录数据
