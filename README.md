# MyDrink

一个功能完整的饮品追踪与健康管理应用 MyDrink，
帮助用户记录每日摄入的咖啡、奶茶、酒和水，分析咖啡因、酒精及水分数据，并提供健康建议。
整体采用 React + Tailwind CSS + Recharts 构建，目前版本的用户数据存储在 localStorage 中。

## 本地运行

1. 先安装 Node.js 20+（安装后会有 `node` 和 `npm` 命令）。
2. 在项目目录执行：

```bash
npm install
npm run dev
```

3. 打开终端输出的本地地址。

## 当前原型功能

- 底部三标签：`记录`、`数据趋势`、`个人设置`
- `记录`页大按钮：`咖啡`、`奶茶`、`酒`、`水`
  - 可查看当天摄入咖啡因、酒精和水分；可查看当天咖啡因、酒精的代谢情况，以及饮水提醒（左右切换）
  - 点击按钮可选杯型、冰度、糖度等并保存
- `数据趋势`页显示：
  - 今日/本周/本月/本年/近7天的饮品占比图
  - 今日/本周/本月/本年/近7天的咖啡因、酒精、饮水量趋势图（左右切换）
  - 历史记录
- `个人设置`页包含个性化设置、个人信息设置、可一键清空全部记录
- 记录自动保存到本地（刷新页面不丢）
