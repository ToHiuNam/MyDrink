# MyDrink

一个面向移动端的饮品记录原型（React + Tailwind CSS）。

## 本地运行

1. 先安装 Node.js 20+（安装后会有 `node` 和 `npm` 命令）。
2. 在项目目录执行：

```bash
npm install
npm run dev
```

3. 打开终端输出的本地地址（通常是 `http://localhost:5173`）。

## 当前原型功能

- 底部三标签：`记录`、`数据趋势`、`个人设置`
- `记录`页大按钮：`咖啡`、`奶茶`
- 点击按钮可选冰度、糖度并保存
- 记录自动保存到本地（刷新页面不丢）
- `数据趋势`页显示：
  - 今日/本周咖啡因趋势切换图
  - 今日或累计饮品占比图
- `个人设置`页可一键清空全部记录
